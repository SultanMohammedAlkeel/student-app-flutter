# وحدة الخدمات الجامعية للطالب

## نظرة عامة

وحدة الخدمات الجامعية هي وحدة متكاملة تم تطويرها لتوفير واجهة موحدة للطالب الجامعي للوصول إلى جميع الخدمات الأكاديمية بسهولة وسلاسة. تتميز الوحدة بتصميم عصري ثلاثي الأبعاد (Neumorphic) مع رسوم متحركة متقدمة وتجربة مستخدم استثنائية.

## الميزات الرئيسية

- **تصميم Neumorphic ثلاثي الأبعاد**: واجهة مستخدم عصرية مع تأثيرات ظل وإضاءة متطورة
- **رسوم متحركة متقدمة**: تأثيرات حركية سلسة تعزز تجربة المستخدم
- **تجاوب كامل**: يعمل بشكل مثالي على جميع أحجام الشاشات (الهواتف، الأجهزة اللوحية، أجهزة الكمبيوتر)
- **دعم وضع الظلام/الفاتح**: تجربة مستخدم مريحة في جميع ظروف الإضاءة
- **دعم كامل للغة العربية**: واجهة مصممة خصيصًا للغة العربية
- **تكامل مع واجهة برمجة التطبيقات**: جلب البيانات من الخادم بشكل متكامل
- **تخزين محلي للمهام**: إدارة المهام وجداول المذاكرة بدون الحاجة للاتصال بالخادم

## الخدمات المتاحة

1. **لوحة الخدمات الرئيسية**: واجهة موحدة للوصول إلى جميع الخدمات مع عرض:
   - معلومات الطالب ونسبة الحضور
   - التنويهات والإشعارات المهمة
   - محاضرات الغد
   - الخدمات الرئيسية والثانوية

2. **الجدول الدراسي**: عرض الجدول الدراسي الأسبوعي الثابت للطالب

3. **المهام وجداول المذاكرة**: إدارة المهام الدراسية وجداول المذاكرة مع:
   - إضافة وتعديل وحذف المهام
   - تصنيف المهام حسب النوع والأولوية
   - عرض المهام حسب التاريخ
   - تتبع إكمال المهام

4. **الدرجات**: عرض درجات الطالب في المقررات المختلفة

5. **المكتبة الإلكترونية**: الوصول إلى الكتب والمراجع الإلكترونية

6. **نماذج الاختبارات**: تصفح وتجربة نماذج الاختبارات الإلكترونية

## كيفية الدمج في المشروع الرئيسي

1. **نسخ الملفات**: نسخ مجلد `student_services_module` إلى مشروع Flutter الرئيسي

2. **إضافة التبعيات**: إضافة التبعيات التالية إلى ملف `pubspec.yaml`:

```yaml
dependencies:
  flutter:
    sdk: flutter
  get: ^4.6.5
  flutter_neumorphic_plus: ^3.3.0
  intl: ^0.18.0
  simple_animations: ^5.0.0+3
  flutter_staggered_animations: ^1.1.1
  shared_preferences: ^2.1.1
  http: ^0.13.6
```

3. **تسجيل المسارات**: إضافة مسارات الوحدة إلى ملف المسارات الرئيسي:

```dart
import 'package:your_app/student_services_module/routes/student_services_routes.dart';

final List<GetPage> routes = [
  // مسارات التطبيق الحالية
  ...
  // إضافة مسارات وحدة الخدمات الجامعية
  ...studentServicesRoutes,
];
```

4. **تسجيل الخدمات**: تسجيل الخدمات المطلوبة في بداية التطبيق:

```dart
import 'package:your_app/student_services_module/utils/storage_service.dart';
import 'package:your_app/student_services_module/utils/api_service.dart';
import 'package:your_app/student_services_module/utils/locale_service.dart';

Future<void> initServices() async {
  // تهيئة خدمة التخزين المحلي
  final storageService = StorageService();
  await storageService.init();
  Get.put<StorageService>(storageService, permanent: true);
  
  // تهيئة خدمة واجهة برمجة التطبيقات
  final apiService = ApiService();
  await apiService.init();
  Get.put<ApiService>(apiService, permanent: true);
  
  // تهيئة خدمة اللغة
  final localeService = LocaleService();
  await localeService.init();
  Get.put<LocaleService>(localeService, permanent: true);
}
```

5. **التنقل إلى الواجهة الرئيسية**: استخدام GetX للتنقل إلى واجهة الخدمات الرئيسية:

```dart
Get.toNamed('/services');
```

## ميزات التصميم

### تصميم Neumorphic ثلاثي الأبعاد

تم استخدام مكتبة `flutter_neumorphic_plus` لإنشاء تصميم ثلاثي الأبعاد متطور مع تأثيرات ظل وإضاءة متقدمة. يتكيف التصميم مع وضع الظلام/الفاتح لتوفير تجربة مستخدم مريحة في جميع ظروف الإضاءة.

### الرسوم المتحركة

تم استخدام مكتبات `simple_animations` و`flutter_staggered_animations` لإضافة تأثيرات حركية سلسة تعزز تجربة المستخدم. تشمل الرسوم المتحركة:

- خلفية متحركة ديناميكية
- تأثيرات ظهور متدرجة للعناصر
- تأثيرات انزلاق وتلاشي
- تأثيرات نقر وتفاعل

### التجاوب مع أحجام الشاشات

تم تصميم الواجهات لتعمل بشكل مثالي على جميع أحجام الشاشات:

- **الهواتف**: تخطيط عمودي مع عناصر مرتبة بشكل متسلسل
- **الأجهزة اللوحية**: تخطيط أكثر اتساعًا مع استغلال المساحة الإضافية
- **أجهزة الكمبيوتر**: تخطيط متعدد الأعمدة مع عرض المزيد من المعلومات

### دعم اللغة العربية

تم تصميم الواجهات خصيصًا للغة العربية مع مراعاة:

- اتجاه النص من اليمين إلى اليسار
- استخدام خط Tajawal المناسب للغة العربية
- تنسيق التواريخ والأوقات بالصيغة العربية
- ترجمة جميع النصوص والرسائل

## وحدة المهام وجداول المذاكرة

تم تطوير وحدة المهام وجداول المذاكرة لتعمل بشكل مستقل عن الخادم باستخدام التخزين المحلي فقط. تتيح الوحدة للطالب:

- إضافة مهام دراسية وجداول مذاكرة
- تصنيف المهام حسب النوع (مهمة/مذاكرة) والأولوية (منخفضة/متوسطة/عالية)
- تحديد تاريخ ووقت استحقاق المهام
- تحديد مدة المذاكرة بالدقائق
- تتبع إكمال المهام
- عرض المهام مرتبة حسب التاريخ (اليوم/غدًا/متأخرة/مستقبلية)
- تعديل وحذف المهام

## الخطوات المستقبلية المقترحة

1. **إضافة إشعارات محلية**: تنبيه الطالب بمواعيد المهام والمحاضرات القادمة

2. **مزامنة المهام مع التقويم**: إضافة المهام إلى تقويم الجهاز

3. **إضافة رسوم بيانية للدرجات**: عرض تقدم الطالب في المقررات بشكل مرئي

4. **تحسين وضع عدم الاتصال**: تخزين المزيد من البيانات محليًا للعمل بدون اتصال بالإنترنت

5. **إضافة ميزات تخصيص الواجهة**: السماح للطالب بتخصيص ألوان وتخطيط الواجهة

## الخلاصة

وحدة الخدمات الجامعية هي حل متكامل يوفر للطالب الجامعي واجهة موحدة للوصول إلى جميع الخدمات الأكاديمية بسهولة وسلاسة. تتميز الوحدة بتصميم عصري وتجربة مستخدم استثنائية مع دعم كامل للغة العربية والتجاوب مع جميع أحجام الشاشات.
